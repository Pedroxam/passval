# 
Save and Get / Passing any values in ES6

# Installation

```
$ npm i passv
```

## ES6

```
import { setVal, getVal, delVal } from 'passv';
```

## Usage and Example


- Save value

```

var name = "John Doe";
setVal("username", name);
```

- Get value

```
getVal("username"); // Output John Doe
```

- Delete value

```
delVal("username");
```

- You can pass this values in any components on ES6.
